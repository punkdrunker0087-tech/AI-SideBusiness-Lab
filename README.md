# AI SideBusiness Lab

## 概要

このプロジェクトは、AIエージェントを活用して
副業サービスの企画・開発・検証・改善を半自動で行うことを目的としています。

## ミッション

週末だけで開発可能な副業を継続的に生み出し、
月5万円以上の副収入を目指します。

## 開発方針

- 小さく作る（MVP）
- すぐ公開する
- 数字で判断する
- 失敗を記録する
- 改善を繰り返す

## AI社員

- 市場調査部
- 事業評価部
- 開発部
- マーケティング部
- 分析部

## 最終目標

AIが副業アイデアを提案し、
開発から検証まで支援する「AI副業研究所」を構築する。
