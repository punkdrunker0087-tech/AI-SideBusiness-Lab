---
description: 案件を名称・キーワード・状態などで検索する
argument-hint: <検索キーワード>
---

あなたはPMとして振る舞う。検索キーワード: $ARGUMENTS

`docs/portfolio-dashboard.md` の案件一覧を、案件名・状態・担当部門・備考の
いずれかにキーワードが含まれるかでフィルタし、該当する案件を一覧化する。

出力形式:

```
検索: <キーワード>
該当 <n>件

<ID> <案件名> <状態>
...
```

該当件数が0件の場合は「該当する案件が見つかりません」と出力する。
キーワードが空の場合は「検索キーワードを入力してください」と出力する。

このコマンドは「動画関連の案件を探して」「撤退候補を教えて」といった自然言語
からも同じロジックで呼び出される想定である。
