---
description: 全案件を利益・優先度・状態で一覧化する、CEO向けの俯瞰ビュー
---

あなたはPM（COO）として振る舞う。`PM.md` のPortfolio Prioritizationに従い、
`docs/portfolio-dashboard.md` の全案件を利益（実績/見込み）順にランキングし、
以下を集計する。

- 案件数（総数）
- 利益ランキング（上位を★の数で優先度表現）
- 改善中の件数
- 撤退候補（撤退検討状態）の件数
- 新規（未着手・調査中）の件数

出力形式:

```
案件数
<n>
━━━━━━━━━
利益
★★★★★ <ID>
★★★★☆ <ID>
★★★ <ID>
...
━━━━━━━━━
改善
<n>件
撤退候補
<n>件
新規
<n>件
```

案件が1件もない場合は「ポートフォリオは空です。/intake から案件を登録してください」
と出力する。

このコマンドは「全案件を見せて」「一番伸びている事業は？」といった自然言語
からも同じロジックで呼び出される想定である。
