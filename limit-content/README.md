# LIMIT CONTENT（仮称） — コンテンツ制作会社OS

LIMIT LAB（LIMIT LAB WORKS）を引き継ぎながら、**コンテンツ制作**を主軸とする
別軸の仮想会社。法人登記はしない（実法人化は`CHARTER.md`の発動条件を満たした
時のみ検討する）。

## LIMIT LABとの関係

```
人間のCEO（同一人物）
 ├── LIMIT LAB WORKS（既存） … プロダクト販売の軸（BOOTH/note/Zenn）
 └── LIMIT CONTENT（本社）  … コンテンツ制作の軸（この会社）
```

- **憲法は共有**: LIMIT LABの`CONSTITUTION.md`（信頼と信用・先義後利・
  利用者第一・感動品質・技術は手段）をそのまま最上位規範として継承する。
  独自の憲法は作らない
- **知見も共有**: `knowledge/`（Decision Log・SOP・Institutional Memory）は
  グループ共通資産として参照・追記する
- **数字は分ける**: 売上・KPIはLIMIT LABと分けて記録し、会社ごとに
  Go/No-Go判断できるようにする
