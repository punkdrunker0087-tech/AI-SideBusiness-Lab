# Toy 0 — 磁石は触るだけで楽しいか（仮説H1）

[開発憲章](../CHARTER.md) の Toy 0 に対応するプロトタイプ。
ゴールなし。ステージなし。パズルなし。**磁力の手触りだけ**を検証する。

## 開き方

1. このリポジトリをZIPでダウンロードして解凍する
   （GitHubのブランチページ → 緑の「Code」ボタン → Download ZIP）
2. Godotのプロジェクトマネージャーで **「インポート」** を押す
3. 解凍したフォルダの `magnet-game/toy0/project.godot` を選ぶ
4. 開いたら **F5**（または右上の▶ボタン）で実行

## 操作

| プレイヤー | 移動 | ジャンプ |
|---|---|---|
| 1P | WASD または ゲームパッド1 左スティック | Space / Aボタン |
| 2P | 矢印キー または ゲームパッド2 | Enter / Aボタン |
| 3P・4P | ゲームパッド3・4 | Aボタン |

一人でも、WASDと矢印キーを両手で操作すれば引力・斥力を体感できる。

| キー | 効果 |
|---|---|
| R | 極性をシャッフル（必ず2人ずつ異極になる） |
| P | **開発者専用**：極性デバッグ表示（頭上に赤/青の球）。テストプレイでは絶対に押さない |
| G | **開発者専用**：Toy 0.5（仮説H6）。「橋を全員で渡ってみて」の目的テキストを表示。初見テストで自由遊びが失速してから点灯し、前後で事故・笑い・行動を比較する |

プレイヤーの体の色（赤・青・緑・黄）は**個人の識別**であり、極性とは無関係。
崖から落ちたら即座に自分のスタート地点へ復帰する。

## 磁力パラメータ（画面右のスライダー）

これがToy 0の実験対象。遊びながらその場で変えて、面白い曲線を探す。

- **強さ**：磁力の基本量
- **減衰指数**：距離で力が弱まる速さ。現実の磁石は2〜4（「何も起きない→突然スナップ」）。小さくすると遠くからじわっと効く
- **力の上限**：近距離での吹っ飛びすぎ防止
- **有効距離**：これより遠いと磁力ゼロ

面白いパラメータの組み合わせを見つけたら、数値をメモして記録すること。

## 測定すること（憲章より）

- 事故が自然に起きるか
- 何度も触りたくなるか
- 「もう一回」が出るか
- **誰も説明していないのに、何を試し始めるか**（最初の自発的実験の記録）
- **指標Z：テスト終了後、誰かが黙ってもう一回始めるか。**
  「もう一回遊ぶ？」とは聞かない。コントローラーを置いたあとの行動だけを見る。
  言葉ではなく行動で測る、ノイズが一番少ない指標

## 棄却条件（実装前に確定済み）

15分遊んで、誰からも「もう一回」が自発的に出なければ棄却。
パラメータ探索を経てもダメなら、磁石という題材ごと再考する。

## 合格条件

初見の別グループでの再現をもって合格。1グループの成功では次へ進まない。
