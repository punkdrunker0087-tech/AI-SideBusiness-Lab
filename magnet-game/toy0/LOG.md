## #1 — 2026-07-16 / 開発者ソロ（キーボード2体操作）

**環境**：初回ビルド。エラーなし。

**観察**：
- 有効距離5前後が「いきなり反発、くっつく」で良い感触

**採用**：
- 有効距離のデフォルトを 8.0 → 5.0 に変更

**この観察から立てた仮説（未検証）**：
- 「じわっと効く」より「何もない→突然バチン」の方が気持ちいい可能性。
  つまり磁力は予兆なしの二値的な挙動の方が、事故（＝笑い）を生みやすいかもしれない。
  → 事故ファースト設計と整合する。Toy 0の残りのセッションと、初見グループで再確認する。

**未記録**：

- 細い橋の上での事故の起きやすさ

## #2 — 2026-07-16 / 開発者ソロ（続き）

**観察**：
- 気持ちいいパラメータのレシピが確定：
  **強さ30 / 減衰指数1.5 / 力の上限45 / 有効距離5**
- 事故は細い橋の上に限らず起きる
- **磁力が弱くても、相互操作で「離れない」状況が生まれる。**
  両者が操作し合うことで、片方だけでは脱出できない吸着状態になる

**採用**：
- デフォルト値を上記レシピに変更（strength 60→30, exponent 2.0→1.5）

**この観察から立てた仮説（未検証）**：
- レシピの構造が面白い：**「境界は鋭く、中は柔らかく」**。
  有効距離5の外はゼロ（突然磁力圏に入る驚き）、圏内は指数1.5の緩い力
  （即死スナップではなく、引きずり合い・綱引きになる）。
  驚きの入口と、操作で抗える中身の組み合わせが気持ちよさの正体かもしれない
- 「離れない」はシステムではなく**人間の相互作用から創発した**状況。
  脱出には相手の協力（またはタイミングを合わせたジャンプ）が要る
  → 「能力は関係性から生まれる」（H5）に接続しうる最初の実測点。
  ただしH5の証拠と呼ぶのはまだ早い。初見グループでの再現待ち

**次に観察したいこと**：
- 「離れない」状態は笑えるか、それともイライラするか
  （笑えるなら維持、イライラなら脱出手段の検討がToy 1の議題になる）

## #3 — 2026-07-16 / 開発者判定

**観察**：
- 「離れない」状態は**笑えるし、イライラする。ゲームとしてはちょうどいい**
- 感触はParty Animalsに近い

**採用**：
- 脱出手段（振りほどきジャンプ等）は**追加しない**。
  創発した吸着状態をそのまま維持する。
  イライラは磁力パラメータではなく、状況の面白さで中和されている

**メモ（位置づけ）**：
- 「Party Animalsに近い」はおもちゃ層の合格サインであって、
  ゲームの独自性の危機ではない。差別化を担うのは極性の非表示・
  仮説ループ（謎層）で、それはToy 2の仕事。
  Toy 0がParty Animalsに似ているのは、むしろ計画通り

**H1の状態**：
- 開発者ソロでは支持に傾いている。
  ただし合格条件は「初見グループでの再現」。
  次のマイルストーンは友人との初カウチセッション

## #4 — 2026-07-21 / 初見グループ（初回）

**観察**：
- 最初の数分は操作性の把握（移動・ジャンプの感覚をつかむ）に使われた
- 最初の事故は**2分後**（30/60/90秒の基準より遅い）
- Gキー（Toy 0.5・目的表示）を押した後、プレイヤーは指示された通りに動くのではなく
  **「何が起こったか」を検証する行動を取った**
- 中身は単純な割に、プレイ時間は長めだった

**解釈（観察と分けて書く）**：
- 事故の遅さ（2分）は、磁力やパラメータの欠陥ではなく**操作学習コスト**が原因の可能性が高い。
  診断表の「30秒で事故が起きない→磁力の強さ・地形の狭さ」という原因候補に、
  **「操作の習熟に時間がかかる」という第三の原因**を追加する必要がある。
  磁力そのものが弱いのか操作学習のせいかは、次回「操作に慣れた後の時間」を
  区切って計測しないと切り分けられない
- Toy 0.5への反応が「素直に従う」でも「無視する」でもなく「検証する」だったのは、
  H6の判定を事故数の単純増減だけで見てはいけないことを示している。
  目的テキストが謎層的な反応（仮説を立てて確かめる）を誘発した点は、
  H3（非対称性は会話を生む）・H6の両方にまたがる示唆として扱う
- プレイ時間の長さ（単純な内容の割に）は、H1・指標Zにとって弱い支持材料
