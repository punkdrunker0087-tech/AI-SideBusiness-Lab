## Qualityデータ源の調査結果（実地検証）

「試せるものは全部試す」で財務データ源を検証した:

- **Yahoo quoteSummary（crumb認証）**: ✅ **登録不要で全25銘柄のROE・粗利率が
  取れた**。資本集約度はDuPont分解（利益率÷ROA=資産回転率、その逆数）で近似。
  → `--live` の4ファクター・ライブランキングとして実装済み（先読みなしの正当な用途）
- **J-Quants API 無料プラン**: 過去2年・12週遅延・要メール登録。**点in-timeに近い
  履歴**が取れる唯一の現実的な無料手段だが、登録が必要
- **EDINET API**: フル履歴だがAPIキー＋XBRL解析が重い
- **SimFin / Alpha Vantage**: 日本株カバレッジ不明

## 今回の到達点と正直な結論

- **設計を実現できた**: β≈0の市場ニュートラルL/Sを、コスト・空売り・分割調整・
  異常値除去まで含めて実装。さらに**Yahoo財務でQualityを取得し、4ファクター版の
  ライブランキング（設計の完全形）を実現**した
- **重要な発見**: `--quality-static`（現在Qを過去に一律適用＝**先読みバイアスありの
  最も有利な条件**）でも、戦略は**依然マイナス（-39%）**だった。
  **ズルをしても勝てない** → 問題はQualityデータの欠如ではなく、
  **ユニバース（25銘柄と少ない）やファクター自体**にある可能性が高い
- **含意**: J-Quantsで点in-time Qualityを苦労して統合しても、この代理ユニバースでは
  勝ち戦略にならない見込み。**優先すべきはQuality調達よりユニバース拡大**
