# AQM-01 v0 ―― Adaptive Quality Momentum（実装版）

設計書 [`MEMO.md`](./MEMO.md) の market-neutral ロング/ショート戦略を、
**現在アクセスできるデータで作れる範囲**まで実装したもの。

> ⚠️ これは設計書の**劣化版（v0）**である。Qualityファクター（財務データ必須）が
> 未実装のため、正確には「Adaptive **Quality** Momentum」ではなく
> 「Adaptive Momentum」。目的は儲けの判定ではなく、**過学習・データ品質・
> コストを正しく扱った上で、価格・出来高部分だけでも市場ニュートラルな
> αの兆しがあるかを測ること**。スコープ詳細は `MEMO.md` 末尾参照。

## 結果の読み方

market-neutral戦略の本当のベンチマークは **TOPIXではなく「現金(0%)」**。
市場に勝つのではなく、**市場と無相関（β≈0）に正のリターン**を出せるかを見る。

```
戦略      : 総リターン -38.5%  Sharpe -0.53 ... β -0.04
TOPIX(B&H): 総リターン +136.3% Sharpe +1.02 ...
対TOPIXベータ = -0.04 → ○ ほぼ市場ニュートラル
判定: × market-neutralな優位は確認できない
```

- **β≈0 は設計通り**（ドルニュートラル構築が機能）。しかし
- **絶対リターンがマイナス** → v0（Quality抜き・25銘柄）ではαは出ていない
- TOPIXが大きく上げていることは無関係（market-neutralは市場方向を取らない）

## 今回の到達点と正直な結論

- **できたこと**: 設計書を実際に動くコードへ落とし込み、市場ニュートラル
  （β≈0）のL/Sを、コスト・空売りコスト・分割調整・異常値除去まで含めて実装した
- **わかったこと**: 価格・出来高だけの劣化版では、この代理ユニバース・期間で
  **市場ニュートラルなαは確認できない**。設計の核である **Quality を入れられるか
  どうかが、この戦略の生死を分ける**可能性が高い
- **断定しないこと**: 25銘柄・単一設定・Quality抜きの結果でAQM-01の可否は決められない
