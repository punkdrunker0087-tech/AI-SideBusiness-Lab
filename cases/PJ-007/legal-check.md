# 法務チェック — PJ-007 頒布レジ

実施: 法務部 / 2026-07-09
対象: 単一HTMLローカルツール（BOOTH販売、製品版¥980以下＋無料版）

## 判定: 法的リスクなし（条件付き・許容範囲内）

法務チェック済みのPJ-006（クリエイター売上台帳、`cases/PJ-006/
legal-check.md`）と同一構成（単一HTML・完全ローカル・AI生成）のため、
同判定を引き継ぎ、本商品固有の差分のみ確認した。

| 論点 | 確認結果・実装要件 |
|---|---|
| レジ・会計機能の位置づけ | 本ツールは利用者自身の私的な記録・計算補助であり、レシート発行・決済処理・金銭の預託は行わない。資金決済法・割賦販売法等の決済規制の対象外 |
| 税務 | PJ-006と同じ表現制限を適用（「記録と集計の支援であり税務助言ではない」を明記） |
| 「完全オフライン」表示 | 外部依存ゼロ（CDN・API・フォント不使用）を実装で担保し、表示と実態を一致させる（景表法・優良誤認の予防） |
| 対象環境の正直表記 | iOSでのローカルHTML起動には制約があるため、動作確認済み環境（PC・Android Chrome推奨）を販売ページ・READMEに明記し、iPhoneでの利用可否を誇張しない |
| データ消失 | 操作ごとの即時保存＋取引の取り消し機能を必須実装とし、免責＋バックアップ推奨を明記 |
| 著作権・商標・AI開示 | PJ-006と同一（自社AI生成・非公式明記・外部ライセンス混入なし） |

## 制作部への実装指示

1. フッターに「税務助言ではない」「データは端末内のみ」「AI生成・
   非公式」の3表示（PJ-006と同文型）
2. 操作ごとの即時保存と、会計の取り消し（在庫復元つき）を必須実装
3. 販売ページに対象環境（PC・Android Chrome推奨／iOSは制約あり）を
   正直に記載
