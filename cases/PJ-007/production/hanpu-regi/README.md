# 頒布レジ — LIMIT LAB WORKS

同人誌即売会・イベント当日の頒布（販売）を、スマホやPCで記録できる
レジツールです。**HTMLファイル1つ**で動作し、**完全オフライン**——
会場の電波が混雑していても止まりません。

## 使い方（当日の朝、3分で準備）

1. ZIPを展開し、`hanpu-regi.html` をブラウザで開く
2. 「📦 頒布物」タブでイベント名と、頒布物（名前・価格・持込数）を登録
3. あとは「🧾 レジ」タブで、売れたらタップ→預かり金をタップ→会計する

## 各タブの説明

| タブ | できること |
|---|---|
| 🧾 レジ | 大ボタンで商品タップ→カート→預かり金（+¥1,000等のボタン）→お釣り自動計算→会計。直前の会計の取り消しも1タップ |
| 📦 頒布物 | 頒布物の登録・持込数の変更（追加搬入対応）。イベント名・日付の設定 |
| 📜 記録 | 会計の履歴。間違えた会計は「取消」で在庫ごと元に戻ります |
| 📊 集計 | 総売上・会計数・頒布点数、頒布物ごとの売れ行きと残部数。CSV書き出し（製品版） |

## 「クリエイター売上台帳」との連携（製品版）

イベントが終わったら、📊集計タブから**売上台帳用CSV**を書き出して、
「クリエイター売上台帳」のCSVインポートで取り込めます。イベント当日は
頒布レジ、日々の記録と確定申告の準備は売上台帳、という分担です。

## 無料版と製品版の違い

| | 無料版 | 製品版 |
|---|---|---|
| 頒布物の登録数 | 3種まで | 無制限 |
| 売上台帳用CSV書き出し | — | ○ |
| レジ・在庫・記録・集計・JSONバックアップ | ○ | ○ |

無料版のデータは製品版にそのまま引き継がれます。

## 動作環境（正直に書きます）

- **推奨: PC（Chrome/Edge/Firefox）または AndroidのChrome**。
  ファイルをタップ/ダブルクリックで開くだけで動きます
- **iPhone（iOS）はローカルHTMLの起動に制約があります。** iOSの
  「ファイル」アプリからのプレビューでは正しく動作しない場合が
  あります。iPhoneでの利用を予定している方は、購入前に無料版で
  動作をご確認ください
