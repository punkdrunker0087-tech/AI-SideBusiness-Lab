# Research Department Prompt
### 「証拠に基づいて判断するAI」— LIMIT LAB WORKS（v1）

副業・個人開発の企画を「なんとなく良さそう」で進めていませんか。
このプロンプト集は、Claude（Claude Code／claude.aiどちらでも）に
「市場調査 → 品質監査 → 8軸スコアリング投資判断」という3段階の
判断プロセスをそのまま実行させるためのものです。

> 本製品はAI（Claude／Anthropic社の生成AIモデル）向けに設計された
> プロンプト集です。Anthropic社による公式提供・公認製品ではありません。

## 含まれるもの

1. **`01-research-prompt.md`** — 実在するURLつきのEvidence（証拠）だけを
   集める市場調査プロンプト。信頼度ランク（A〜E）付き
2. **`02-validation-prompt.md`** — 市場調査結果を鵜呑みにせず、
   チェックリストで品質監査するプロンプト（PASS/REWORK判定）
3. **`03-evaluation-prompt.md`** — PASS済みの調査結果を8軸で採点し、
   5段階（Strong Go〜No-Go）で投資判断するプロンプト

## こんな人におすすめ

- 副業・個人開発の企画を、勘ではなく数字で判断したい人
- AIに市場調査をさせても「それらしい話」ばかりで裏付けが薄いと
  感じたことがある人
- 複数の事業アイデアを同じ基準で比較したい人

## 使い方

1. `01-research-prompt.md`の本文をコピーし、案件名・概要を書き足して
   Claudeに貼り付ける
2. 出力結果を`02-validation-prompt.md`に貼り付けて監査する
3. PASSしたら`03-evaluation-prompt.md`に貼り付けてスコアリングする

コードを書く必要はありません。3つのプロンプトを順番にコピー＆
ペーストするだけです。
