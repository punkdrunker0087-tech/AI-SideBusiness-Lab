# 特定商取引法に基づく表記（テンプレート）

日本国内向け販路（BOOTH・noteなど）でPJ-002の成果物を販売する場合、
以下の項目を出品者本人の実情報で埋めて使用してください。本テンプレート
自体には実在の個人情報を含みません。

| 項目 | 記入内容 |
|---|---|
| 販売業者 | （氏名または屋号を記入） |
| 運営責任者 | （氏名を記入） |
| 所在地 | （請求があれば遅滞なく開示する、とする場合はその旨を記載） |
| 連絡先 | （メールアドレス等。請求があれば遅滞なく開示する、とする場合は
  その旨を記載） |
| 販売価格 | 各商品ページに記載の価格（税込） |
| 商品代金以外の必要料金 | なし（決済手数料はプラットフォーム規約に準ずる） |
| 支払方法 | 各プラットフォームの提供する決済手段 |
| 支払時期 | 購入手続き完了時 |
| 引き渡し時期 | 決済完了後、即時ダウンロード可能 |
| 返品・キャンセル | デジタルコンテンツの性質上、購入後の返品・返金は
  原則不可（プラットフォーム規約に準ずる） |

個人事業主が所在地・連絡先の即時公開を避けたい場合、「請求があれば
遅滞なく開示します」という記載が一般的に用いられているが、最終的な
記載内容・可否は各プラットフォームの規約と最新の法令に従って出品者
本人が判断すること。本テンプレートは法的助言ではない。
