# AI Company OS Starter（無料サンプル版）

> 「ひとり社長のためのAI経営OS」— An AI Executive Operating System

Claude Code上で「小さな会社」を運営するAI会社OSの、無料スターター版です。
思いついた副業・プロジェクトのアイデアを「管理された案件」として受付・

## できること

- `/intake` — アイデアを案件として登録（ID採番・上限3件のポートフォリオ管理）
- `/status` — 案件の状態確認（停滞案件の検出つき）
- Portfolio Dashboardによる「唯一の真実の源」方式の案件管理

## セットアップ（3分）

1. このZIPを展開する
2. `CLAUDE-STARTER.md` を、あなたのプロジェクトのルートに `CLAUDE.md`
   という名前で置く
3. `portfolio-dashboard.md` を同じ場所に置く
4. `commands/` 内の2ファイルを `.claude/commands/` に置く
5. Claude Codeで「/intake 週末にやりたい副業のアイデア」と話しかける

コードを書く必要はありません。すべてMarkdownファイルです。
