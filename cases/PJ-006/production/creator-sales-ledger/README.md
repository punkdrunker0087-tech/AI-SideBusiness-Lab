# クリエイター売上台帳 — LIMIT LAB WORKS

BOOTH・note・Skebなどで活動するクリエイター・ひとり社長のための、
売上・経費・KPI記録ツールです。**HTMLファイル1つだけ**で動作し、
Excelもアカウント登録も不要。データはあなたの端末の中にだけ保存されます。

## 使い方（30秒で開始）

1. `creator-sales-ledger.html` をダブルクリックで開く（それだけです）
2. 「📝 記録」タブで、売れたもの・使った経費・フォロワー数などを入力
3. 「📊 ダッシュボード」で今月の利益と推移グラフを確認
4. 「📅 年間集計」で確定申告の準備に使える年間サマリーを確認

## 各タブの説明

| タブ | できること |
|---|---|
| 📝 記録 | 収入（日付・プラットフォーム・品名・金額）、経費（カテゴリ・金額）、KPI（フォロワー数など）の入力と削除 |
| 📊 ダッシュボード | 今月の収入・経費・利益（前月比つき）、直近12ヶ月のグラフ、KPIの推移グラフ |
| 📅 年間集計 | 年ごとの合計、月別内訳、プラットフォーム別・経費カテゴリ別の集計、CSV書き出し（製品版） |
| 💾 データ | JSONバックアップの書き出し/読み込み、プラットフォーム・経費カテゴリ・KPI指標の追加/削除（自分仕様にカスタマイズできます） |

## 大切なお知らせ

- **データはお使いの端末のブラウザ内にのみ保存されます。** 外部への
  送信は一切ありません。ただし、ブラウザの履歴・サイトデータの削除や
  端末の故障で失われることがあるため、「💾 データ」タブから**定期的に
  バックアップを書き出すことを強くおすすめします**
- 本ツールは売上・経費・KPIの**記録と集計を支援する**ものであり、
  **税務助言・申告代行を行うものではありません**。確定申告の内容は
  税理士等の専門家にご確認ください

  公式製品・公認製品ではありません
