# クリエイター売上台帳 — LIMIT LAB WORKS

BOOTH・note・Skebなどで活動するクリエイター・ひとり社長のための、
売上・経費・KPI記録ツールです。**HTMLファイル1つだけ**で動作し、
Excelもアカウント登録も不要。データはあなたの端末の中にだけ保存されます。

## 使い方（30秒で開始）

1. `creator-sales-ledger.html` をダブルクリックで開く（それだけです）
2. 「📝 記録」タブで、売れたもの・使った経費・フォロワー数などを入力
3. 「📊 ダッシュボード」で今月の利益と推移グラフを確認
4. 「📅 年間集計」で確定申告の準備に使える年間サマリーを確認

## 各タブの説明

| タブ | できること |
|---|---|
| 📝 記録 | 収入（日付・プラットフォーム・品名・金額）、経費（カテゴリ・金額）、KPI（フォロワー数など）の入力と削除。種類・月・キーワードでの絞り込み |
| 📊 ダッシュボード | 今月の収入・経費・利益（前月比つき）、手取り概算（製品版）、目標達成率バー、直近12ヶ月のグラフ、KPIの推移グラフ |
| 📅 年間集計 | 年ごとの合計、月別内訳、プラットフォーム別・経費カテゴリ別の集計、CSV書き出し・印刷用レポート（製品版） |
| 💾 データ | JSONバックアップ、目標設定、手数料設定（製品版）、CSVインポート（製品版）、プラットフォーム・経費カテゴリ・KPI指標の追加/削除（自分仕様にカスタマイズできます） |

## 製品版だけの機能

- **CSVインポート**: BOOTH等からダウンロードした売上CSVを読み込み、
  収入として一括登録（列の対応付けは画面で選択。重複は自動で除外）
- **手数料・手取り計算**: プラットフォームごとに手数料率（例: 5.6%＋
  45円）を設定すると、額面と手取り概算の両方を表示
- **印刷用レポート**: 年間レポートをそのまま印刷／PDF保存できる
  白背景レイアウトで出力
- **CSV書き出し**: Excel・会計ソフトへの取り込み用
