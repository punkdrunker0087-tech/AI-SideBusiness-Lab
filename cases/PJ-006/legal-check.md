# 法務チェック — PJ-006 クリエイター売上・KPI記録ツール

実施: 法務部 / 2026-07-08
対象: 単一HTMLローカルツール（BOOTH販売、¥980以下＋無料版）

## 判定: 法的リスクなし（条件付き・許容範囲内）

| 論点 | 確認結果・実装要件 |
|---|---|
| 税理士法 | **「確定申告対応」「申告書作成」等の表現を使わない。**「売上・経費の記録と集計を支援するツールであり、税務助言・申告代行を行うものではない」旨をアプリ内とBOOTH説明文の両方に明記する（税務相談・申告書類作成の代行は税理士の独占業務のため、誤認させる表現を排除） |
| 個人情報保護 | データは購入者の端末（localStorage）のみに保存され、開発者を含む外部への送信は一切ない。この事実を明記する（プライバシー訴求と表裏一体） |
| 著作権・商標 | 全コードは自社（AI生成）オリジナル。外部ライブラリ・CDN・フォントを使用しない完全自己完結構成のため、第三者ライセンスの混入なし |
| AI生成開示 | 既存商品と同じ方針でAI（Claude）による設計・生成である旨を明記 |
| 特定商取引法 | ショップ全体の表記が既に存在。追加対応不要 |
| 免責 | データ消失（ブラウザのデータ削除・端末故障等）に備え、エクスポート/バックアップ機能を提供し、定期バックアップを利用者に推奨する免責文を同梱する |

## 制作部への実装指示

1. アプリ内フッターと同梱READMEに「本ツールは税務助言を行うものでは
   ありません」を必ず表示する
2. 「データはこの端末のブラウザ内にのみ保存されます」をアプリ内に表示する
3. JSONバックアップ（書き出し・読み込み）を無料版含む全バージョンに
   搭載する（データ消失リスクは信頼と信用に直結するため、有料機能に
   しない）
