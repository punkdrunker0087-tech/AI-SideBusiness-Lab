# LIMIT LAB 会社設立プロジェクト

LIMIT LABの事業・ブランドを引き継ぎながら、別軸（AI副業研究所 = AI-SideBusiness-Lab）で展開するための会社設立準備ドキュメント。

## このプロジェクトの進め方

「AIにできること」と「AIにできないこと（人間＝代表者本人がやること）」を明確に分離し、
できることはAIが下準備・ドラフト作成まで進め、できないことはチェックリスト化して本人が実行する。
