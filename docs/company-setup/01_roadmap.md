# 会社設立ロードマップ（2026年時点）

日本で会社を設立するための標準的な流れと費用。LIMIT LABの引き継ぎ形態が決まり次第、このロードマップに引き継ぎ用のステップを追加する。

## ステップ一覧

### フェーズ1: 設計（AIが大部分を支援可能）

1. **会社形態の決定** — 株式会社 or 合同会社
2. **基本事項の決定** — 商号、事業目的、本店所在地、資本金、事業年度、役員構成
3. **定款の作成** — 電子定款にすれば印紙代4万円が不要

### フェーズ2: 法的手続き（本人の実行が必須）

4. **定款認証**（株式会社のみ。合同会社は不要）
   - 公証役場で認証。ウェブ会議での認証も可能
   - 手数料: 資本金に応じて3万〜5万円。ただし2024年12月から、**資本金100万円未満 + 発起人が自然人3名以下 + 発起人が全株式を引受 + 取締役会非設置** の条件を満たせば **1万5,000円** に引き下げ
5. **資本金の払い込み** — 発起人個人の銀行口座へ振込
6. **設立登記申請** — 法務局へ申請。オンラインなら「法人設立ワンストップサービス」（マイナポータル）で定款認証〜登記〜税務届出をまとめて申請可能
   - 登録免許税: 株式会社は **15万円**（または資本金×0.7%の高い方）、合同会社は **6万円**（同）
   - 自治体の特定創業支援等事業の証明書があれば **半額**（株式7.5万円 / 合同3万円）になる場合あり

### フェーズ3: 設立後の届出（AIが書類ドラフト支援、提出は本人）

7. 税務署: 法人設立届出書、青色申告承認申請書、給与支払事務所等の開設届出書
8. 都道府県税事務所・市区町村: 法人設立届出書
9. 年金事務所: 健康保険・厚生年金の新規適用届（役員報酬を出す場合は必須）
10. 銀行: 法人口座開設
11. その他: 法人印鑑作成、会計ソフト導入、（必要なら）許認可申請

## 費用まとめ

| 項目 | 株式会社 | 合同会社 |
|---|---|---|
| 定款印紙代（電子定款なら0円） | 4万円 → **0円** | 4万円 → **0円** |
| 定款認証手数料 | 3〜5万円（条件を満たせば1.5万円） | **不要** |
| 登録免許税 | 15万円〜 | 6万円〜 |
| **合計（電子定款の場合）** | **約16.5万〜20万円** | **約6万円** |

※ 創業支援証明書で登録免許税が半額になれば、株式会社 約9万円〜 / 合同会社 約3万円〜。
