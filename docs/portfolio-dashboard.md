# portfolio-dashboard.md — Portfolio Dashboard

`/intake` `/route` `/portfolio` `/review` `/status` `/search` などのコマンドは
すべてこのファイルを参照・更新する。手動での直接編集は避け、原則として
コマンド経由（PMの操作）でのみ更新する。
