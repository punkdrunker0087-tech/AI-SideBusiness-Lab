# 会社ルール固定: 戦略判断の前に「BOOTHアクセス数」の測定を必須とする

日付: 2026-07-20
決定: CEO承認。**大きな戦略判断（チャネルへの全張り・価格改定・高単価
リパッケージ・撤退等）を下す前に、BOOTHアクセス数（＝流入の実数）が
測定されていることを前提条件とする。** 未測定のまま大きく賭けない。

  メトリクスにもDecision Logにも一度も出てこない（7/8ログで「依頼する」
  と書いたきり未取得）

## なぜ前提条件にするか（打ち手が正反対に分岐するから）
- **views ≈ 0 → reach問題**: 通行人がいない。打ち手は「届け方（チャネル・
  検索資産・告知先）」
- **views > 0 かつ DL = 0 → offer/商品問題**: 通行人はいるが刺さらない。
  打ち手は「商品・価格・ページ・訴求」
- この2つは対策が真逆。**どちらか分からないまま「集客に全張り」するのは、
  病名を確定せず手術するのと同じ**（憲法「数字で判断する」・CLAUDE.md
  §11の徹底）
