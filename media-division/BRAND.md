# LIMIT LAB ブランド戦略（Media Division）

制定: 2026-07-20（CEO指示を明文化）

研究プロトコル（v2026.1・凍結済み）とは別レイヤーの文書。プロトコルが
「何を検証し、どう記録するか」を定めるのに対し、本書は「どう見られるか、
どう信頼を積むか」を定める。

---

## ブランドの核（一文）

**「AIと人間は、どこまで一緒に会社を運営できるのかを実験し、
その過程を公開する研究所。」**

- LIMIT LABは「AI会社」ではなく、メディアブランドとして見られることを目指す
- 5年後の到達点: 視聴者が「ここはAIツール屋ではない。**AI運営の実験結果が
  集まる場所だ**」と認識する状態。「LIMIT LAB」という名前自体が信頼のラベルになる

## コンテンツの4役割

すべてのコンテンツは、公開前にこの4分類のどれかに割り当てる。
役割のないコンテンツは作らない。順番は **ブランド → 信頼 → 商品**。

| 役割 | 目的 | 例 |
|---|---|---|
| ① Trust（信頼） | 「本当に検証している」と思ってもらう | 「今日のOSはどこで現実とぶつかったか」/ AIの失敗記録 / 介入率の推移 |
| ② Reach（認知） | ショートで新しい人に知ってもらう | AI社長の朝礼 / 30秒実験 / AIニュース検証 |
| ③ Depth（深い理解） | 背景を説明し、ファンを研究の同伴者にする | EXP-005週報 / Postmortem / Research Log（note・長尺） |
| ④ Revenue（収益） | 信頼の上で初めて商品を置く | AIツール / テンプレート / OS・SOP / 講座 |

位置づけ: **動画は入口。noteは研究ノート。テンプレートは成果物。
AIツールは実験で生まれた副産物。**

## トーン: 「静かな研究所」

煽らない。記録する。「AIで100万円！」の対極に立つ。

- **配色**: ネイビー基調（#0b0e14）/ 白（#f2f4f8）/ 黄アクセント（#ffd166）。
  補助色としてteal（#4cc9b0）を現行テンプレートから継続（変更する場合は
  Brand Guardianチェックを通す）
- **意匠**: 実験番号 / 観測ログ / シンプルな図 / 研究ノートの体裁
- **語り口**: 断言しない（憲法第4条）・数字で語る・失敗を隠さない
- この統一感を崩す施策は、短期的に数字が見込めても採用しない

## Brand Guardian（AI編集者）

投稿前の最終チェックを担うAI部門。制作はしない。編集者に近い役割。
人間のブランド判断（L3）の**前段フィルタ**であり、人間の判断を代替しない。

**公開前チェックリスト（全項目Yesで提出可）:**

- [ ] 世界観に合うか（静かな研究所か。煽っていないか）
- [ ] Evidence Firstか（実測に基づくか。検証していないことを断言していないか）
- [ ] 誇張していないか（数字・効果の盛りがないか）
- [ ] AIを神格化していないか（AI万能論・AI崇拝になっていないか）
- [ ] キャラクターが崩れていないか（AI社長はキャラであることを隠していないか）

チェック結果は各日の`publish.md`に記載する。1項目でもNoなら修正してから提出
（この修正はL2介入ではなくAI内部の品質工程としてカウントする）。

## 自動化の設計思想: 「人間が価値を生まない作業をゼロに」

「全部AI」ではない。創業者の担当を **ブランド・実験テーマ・世界観・倫理・
終了判断**（=L4領域）に純化していくことが目標。

```
Research → 企画 → 台本 → 動画生成 → 字幕 → 分析 → 改善案   : AI（現状すでにL0）
承認・投稿                                              : 人間（現状L3）
L4判断（ブランド・テーマ・倫理・終了）                        : 常に人間
```

### 投稿の自動化（段階的自律・将来の実験候補）

CEO構想の最終形では投稿予約もAIが担うが、**現時点では採用しない**。理由:

1. AI CEO憲法第5条・CHARTER第4条（公開は人間のみ）と矛盾する。
   改定はL4事項かつバージョン改定（v2026.2以降）に該当する
2. 技術的にも、YouTubeへの投稿はアカウント認証情報が必要であり
   no-credentials-boundaryに抵触する（人間がAPI連携を設定する作業が前提）
3. EXP-005は「人間の承認・投稿」を含む現行OSを測定中。実験途中の変更は
   測定対象そのものを変えてしまう

**導入条件（自動運転のレベル引き上げと同型で、実験として行う）:**
EXP-005完了後のPostmortemで、L2介入率が十分低く（目安: 週次15%未満が
2週間継続）、Brand Guardianチェックの有効性が確認された場合に、
「AI投稿予約 + 人間の事後拒否権（公開前レビュー時間枠）」を
新しい実験（EXP-00N）として設計し、憲法改定とともにL4に諮る。
