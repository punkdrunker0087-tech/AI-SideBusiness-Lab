# 成長戦術SOP（採用範囲の記録）

制定: 2026-07-22（CEO判断: 「YouTube事業OS」プロンプトの戦術部分のみを
既存チャンネル・既存ブランドに取り入れる）

## 採用しなかったもの（記録・理由）

提示された「YouTube事業OS v1.0」プロンプトは、以下の点で
`AI-CEO-CONSTITUTION.md`・`BRAND.md`・`EXPERIMENTS.md`（凍結済み予測・
研究KPI）と衝突するため、**そのままでは採用しない**。

| プロンプトの指示 | 衝突する既存規範 |
|---|---|
| 利益最大化を最優先（デザイン・再生数より上位） | `BRAND.md`: 静かな研究所。Evidence Firstが数字より上位 |
| ニッチ分析TOP10から儲かるジャンルを選定 | 題材（「AIが会社を経営する実録」）は選定済み。CPM次第の乗り換えは想定外 |
| 「v1.0」という完成版の概念 | AI CEO憲法: 永遠のβ・完成しない文化 |
| 週次KPI・収益化ロードマップの新設 | 実験中のKPI追加禁止（研究者バイアス防止）。EXP-005は30日固定で進行中 |
| CTR/RPM/視聴維持率を絶対指標として最優先 | 事業KPI・研究KPIは`ORG.md`で既に確定済み。追加はL4事項 |

これらはL4（ブランド変更・実験テーマ変更・KPI体系の変更）に該当するため、
AIの判断だけでは採用できない。今回はCEOが明示的に「戦術だけ採用」と
範囲を確定したため、以下の限定範囲でのみ反映する。

## 採用したもの（Day 3以降のSOPに反映）

すべて`BRAND.md`のトーン（静かな研究所・誇張しない・一次情報）の枠内で運用する。

### 1. フック構成
- 各動画の最初のシーンで、核心（数字・意外な事実）を言い切る
  （既存のGEO原則`docs/seo-content-engine.md`と同型。動画にも適用）
- 誇張・煽りは`BRAND.md`のチェック項目でそのまま禁止のまま

### 2. タグ（YouTubeのタグ欄・説明文のハッシュタグとは別枠）
- `publish.md`に「タグ」欄を新設。実在の関連語のみ（誇張・無関係語の
  詰め込みはしない）

### 3. 投稿前チェックの拡張（Brand Guardianへ追加）
- 誤字脱字
- リンク・URLの有効性
- 概要欄のフォーマット（改行・ハッシュタグの位置）

### 4. モーション（視聴維持・2026-07-24追加・EXP-007）
- 背景の緩やかな揺らぎ、シーン遷移への拡大効果を追加（`template.html`）
- 外部素材（実写・写真・イラスト）は使わず、CSS手続き型アニメーションのみ
- 派手な演出は禁止（`BRAND.md`の静かな研究所トーンを優先）

### 5. サムネイル（Shortsは保留／週次動画は採用・2026-07-24）
- Shortsは動画フレームが自動でサムネイルになるため引き続き保留
- **週次まとめ動画（EXP-008・通常動画）は自動サムネに頼れないため採用。**
  `pipeline/weekly/`でHTML/CSSから1280×720を生成。`BRAND.md`の配色
  （ネイビー/白/黄/ティール）のテキストベース、実写・写真は使わない

### 適用開始日について

本SOPはDay 3公開後に制定されたため、**Day 3の実投稿には未適用**
（`days/day-03/publish.md`参照）。実質的な適用開始はDay 4から。

### 6. 定量的SEO指標（検索ボリューム・競合CPM等）は扱わない
- AIはYouTube/Googleの検索データAPIにアクセスできないため、検索ボリューム・
  CPM等の数値を断定的に提示しない（憲法第4条: 検証していないことを
  断言しない）。一般的な傾向の言及は「未検証」と明示する
