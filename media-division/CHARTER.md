# LIMIT LAB Media Division 仮想定款（CHARTER）

実在の法人ではないが、実法人の定款に相当する「会社の根本規則」をここに定める。
実法人化する際は、本CHARTERをそのまま実定款のドラフト原案として転用する。

制定: 2026-07-19（CEO決定）

---

## 第1条（商号）

当部門は LIMIT LAB Media Division と称する（2026-07-19 CEO決定: ブランドはLIMIT LABに統一し、別会社名を立てない）。

## 第2条（目的）

当会社は次の事業を行うことを目的とする。

1. 文章・画像・音声・映像等のデジタルコンテンツの企画・制作・編集・配信・販売
2. コンテンツ制作に関するテンプレート・ツール・ノウハウの販売
3. AIを活用したコンテンツ制作手法の研究・記録・公開
4. 前各号に附帯関連する一切の事業

## 第3条（LIMIT LABからの引き継ぎ）

1. 当会社はLIMIT LAB WORKSの以下を継承する。
   - `CONSTITUTION.md`（LIMIT LAB憲法）— 最上位規範としてそのまま採用
   - `knowledge/` 配下の組織知（Decision Log・SOP・Institutional Memory 等）
   - 経営原則: 意思決定は人間のCEOに帰属する / 数字で判断する /
     失敗は資産として記録する / 利益は目的ではなく存続の条件
2. 案件パイプライン（発掘フェーズ → 実行フェーズ）と部門の仕組みは
   LIMIT LABのものを再利用する（Reuse First）。当会社専用の部門は、
   必要が数字で証明されるまで新設しない。

## 第4条（機関）

1. 当会社の法的な代表者・最終責任者は人間（代表）1名とする。
2. 「AI CEO」は当会社の運営コンセプトであり、コンテンツ世界における
   キャラクターである。役職・法的地位ではない（詳細は`ORG.md`）。
   AI CEOは日常運営（テーマ決定・制作指示・改善提案）を担うが、
   その効力は人間の承認を経て初めて対外的に発生する。
3. AIは調査・企画・制作・分析・ドラフト作成を行うが、以下は人間（代表）
   のみが行う。
   - 最終承認・投稿（公開ボタンは人間だけが押す）
   - 投資・継続・撤退の判断
   - 法的確認・ブランド判断・契約行為
   - 金銭の移動、アカウント認証情報の取り扱い（no-credentials-boundary準拠）

## 第5条（事業年度・レビュー）

LIMIT LABのレビュー階層（Morning Brief / Weekly Review / Performance Review /
Annual Reflection）に相乗りし、当会社の数字は独立の項目として記録する。

## 第6条（実法人化の発動条件）

以下のいずれかを満たした場合、CEOは実法人化（法人登記）の検討を開始する。
検討時は `../docs/company-setup/` のロードマップに従う。

1. グループ合計の月次利益が **3ヶ月連続で10万円** を超えたとき
   （法人維持コスト: 法人住民税均等割 年約7万円 + 税理士費用等を
   利益で吸収できるライン）
2. 法人格がないことで失注する取引（企業案件等）が実際に発生したとき
3. 事業所得が拡大し、税制上法人が有利になるとCEOが判断したとき

※ 条件を満たすまで登記はしない。売上0円での登記費用支出は
「数字で判断する」原則に反するため禁止する。

## 第7条（改定）

本CHARTERの改定はCEOの決定による。改定は `knowledge/Decision Log/` に
記録する。
